## 快速开始

### 第一步：安装并启动

1. 下载 FlowMate 安装包（FlowMate_x.x.x_x64-setup.exe）
2. 运行安装包，按照提示完成安装
3. 安装完成后，FlowMate 会自动在后台启动

### 第二步：确认运行状态

安装后，FlowMate 会在系统托盘区域显示图标。

- 托盘图标正常显示 = 服务正在运行
- 点击托盘图标可以打开设置页面

### 第三步：生成日报

**方式一：使用快捷键**
- 默认快捷键：`Ctrl + Shift + D`
- 按下快捷键，日报会自动生成并复制到剪贴板

**方式二：点击托盘菜单**
1. 右键点击托盘图标
2. 选择「生成日报」
3. 日报生成后会显示预览窗口

## 功能说明

### 自动采集

FlowMate 会自动在后台采集以下信息：

| 采集类型 | 说明 |
|---------|------|
| 剪贴板 | 自动记录复制的文本内容 |
| 窗口标题 | 记录你当前使用的应用程序和窗口 |
| 文件操作 | 记录打开/修改过的文件（需开启监听） |

这些数据会保存在本地数据库中，仅用于生成日报，不会上传到任何服务器。

### 日报生成

生成日报有两种模式：

**AI 云端生成（推荐）**（目前不需要配置）
- 调用云端 AI API 生成结构化日报
- 支持多种 AI 模型
- 需要配置 API 地址和密钥

**本地兜底生成**
- 当无法连接云端 API 时自动启用
- 生成简化版日报（统计各项活动数量）
- 无需网络连接

### 系统托盘

托盘图标右键菜单包含：

- **生成日报** - 手动触发日报生成
- **设置** - 打开设置页面
- **暂停采集** / **恢复采集** - 控制数据采集开关
- **退出** - 完全退出程序

### 全局快捷键

默认快捷键：`Ctrl + Shift + D`

## 设置页面详解

### 数据采集

**启用采集**
- 开关控制：开启或关闭所有数据采集功能
- 关闭后，FlowMate 不会记录任何工作活动

**剪贴板采集**
- 自动记录所有复制的文本
- 同一内容不会重复记录

**窗口标题采集**
- 记录当前使用中的应用窗口
- 支持白名单模式，只记录特定应用

### 定时提醒

**启用提醒**
- 开启后，每天会在指定时间收到生成日报的提醒通知

**提醒时间**
- 设置每天希望收到提醒的时间
- 默认时间：下午 6:00

### 文件目录监听

**启用监听**
- 开启后，会记录指定目录下的文件操作

**监听目录**
- 点击「添加目录」选择要监听的文件夹
- 可以添加多个目录
- 支持移除已添加的目录

### 应用白名单

如果只希望记录特定应用的工作情况：

1. 点击「添加应用」
2. 选择要添加的应用程序（.exe 文件）
3. 添加后，只有这些应用的活动会被记录
4. 如果白名单为空，则记录所有应用

### AI 配置（目前不需要用户自己配置）

 

### 快捷键设置

- 点击输入框，然后按下你想要的快捷键组合
- 建议使用 Ctrl、Shift、Alt 与字母键的组合
- 避免与常用软件的快捷键冲突

## 常见问题

### Q: FlowMate 会不会收集敏感信息？

不会。FlowMate 采集的所有数据都存储在本地电脑的数据库中，不会自动上传到任何服务器。只有当你主动点击「生成日报」时，才会将当天的活动数据发送给 AI API 用于生成日报。

### Q: 如何停止 FlowMate 的所有采集？

有两种方式：

1. **临时停止**：右键托盘图标 → 选择「暂停采集」
2. **完全退出**：右键托盘图标 → 选择「退出」

### Q: 快捷键不起作用怎么办？

1. 确保 FlowMate 正在运行（托盘图标可见）
2. 检查快捷键是否与其他软件冲突
3. 可在设置中更换为其他快捷键组合

### Q: AI 生成日报失败怎么办？

FlowMate 内置了本地兜底生成模式，当云端 API 不可用时会自动使用本地模式生成简化版日报。

如果希望使用 AI 生成：
- 检查 API 地址和密钥是否配置正确
- 确认网络连接正常
- 尝试切换到免费模型

### Q: 日报内容不准确怎么办？

1. 检查白名单设置，确保相关应用已被添加
2. 确认采集功能已开启
3. 检查数据采集时间范围（仅统计当天数据）
4. 可以在预览窗口中手动编辑日报内容

### Q: 如何查看今天采集了多少数据？

1. 右键托盘图标 → 选择「设置」
2. 页面会显示今日已采集的记录条数
3. 或者打开日报预览查看详细统计

### Q: FlowMate 占用系统资源多吗？

不多。FlowMate 采用轻量级设计：
- CPU 占用率 < 1%
- 内存占用约 50-100MB
- 对日常办公几乎没有影响

### Q: 如何卸载 FlowMate？

1. 打开 Windows 设置
2. 进入「应用」→「已安装的应用」
3. 找到 FlowMate
4. 点击「卸载」
