# FlowMate

自动化工作日报生成工具。

## 功能特点

- 自动采集工作活动（剪贴板、窗口标题、文件操作）
- AI 智能生成结构化日报
- 全局快捷键一键生成
- 定时提醒
- 本地数据存储，保护隐私
